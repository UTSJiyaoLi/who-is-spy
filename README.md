# 谁是卧底助手

一款专为聚会场景设计的「谁是卧底」游戏辅助工具，支持 iPhone 浏览器直接打开使用，无需安装。

## 功能特性

- **词语生成**：内置 60+ 组精选词语，覆盖饮品、美食、职业、影视等 10+ 分类
- **人数设置**：支持 3~20 人，可自由配置卧底人数
- **角色系统**：支持平民、卧底、白板、裁判四种角色
- **裁判模式**：裁判专用面板，密码保护，可查看所有人身份并标记出局
- **身份分配**：按住查看身份，松手立即隐藏，防偷看设计
- **发言计时器**：30/60/90/120 秒可选，支持倒计时提醒
- **随机发言顺序**：自动生成并支持重新排序
- **投票统计**：每轮记录各玩家得票数，自动计算最高票出局
- **词库管理**：支持添加自定义词语
- **历史记录**：自动保存每局游戏结果

## 技术栈

- Vue 3 + Composition API
- Pinia 状态管理
- Vue Router
- Vite PWA

## 本地开发

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

构建产物在 `dist/` 目录，可部署到任何静态托管服务（Vercel、Netlify、GitHub Pages 等）。

## iPhone 添加到主屏幕

1. 用 Safari 打开应用链接
2. 点击底部分享按钮
3. 选择「添加到主屏幕」
4. 即可像原生 App 一样全屏使用

## 使用流程

1. 点击「开始游戏」进入设置
2. 配置人数、卧底数、是否启用白板/裁判
3. 选择词库分类或自定义词语
4. 生成身份后，玩家轮流按住屏幕查看身份
5. 裁判可进入裁判面板管理游戏进程
6. 使用游戏工具进行发言计时和投票统计

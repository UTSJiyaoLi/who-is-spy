# 谁是卧底助手 · 多人联机版

一款专为聚会设计的「谁是卧底」实时联机游戏工具。每人用自己的手机加入，无需传递手机，裁判统一管理身份和进程。

在线体验 👉 [https://who-is-spy.onrender.com](https://who-is-spy.onrender.com)

---

## 功能介绍

- **创建房间**：一键创建游戏房间，自动生成 4 位邀请码
- **扫码加入**：房间页面显示二维码，好友扫码即可加入
- **自动分配身份**：系统随机分配平民 / 卧底 / 白板身份
- **裁判模式**：创建者自动成为裁判，可查看所有人身份并标记出局
- **防偷看设计**：查看身份时按住屏幕显示，松手立即隐藏
- **游戏工具**：发言计时器、随机发言顺序、投票统计
- **丰富词库**：内置 10+ 分类 60+ 组精选词语

---

## 使用步骤

### 1. 创建房间

打开链接 → 点击「创建房间」→ 设置人数和规则 → 成为裁判

### 2. 邀请好友

将屏幕上的**二维码**或**4 位邀请码**分享给朋友

### 3. 加入房间

好友扫码或输入邀请码 + 昵称即可加入

### 4. 开始游戏

所有人到齐后，裁判点击「开始游戏」，系统自动分配身份

### 5. 查看身份

每位玩家在自己的手机上按住屏幕查看专属身份

### 6. 游戏进行

使用「游戏工具」进行发言计时和投票，裁判在「裁判面板」标记出局玩家

## 游戏规则

### 角色说明

| 角色 | 人数 | 特点 |
|------|------|------|
| **裁判** | 1 人 | 创建房间者，可看到所有人身份，负责主持游戏 |
| **平民** | 多数人 | 拿到相同的词语 |
| **卧底** | 1~2 人 | 拿到不同的词语，需要隐藏身份 |
| **白板** | 0~1 人 | 没有词语，需要根据描述推理（可选） |

> 裁判不计入玩家总数。

### 游戏流程

1. 每人用一句话描述自己的词语，**不能直接说出词语**
2. 一轮描述结束后，大家讨论并投票选出最像卧底的人
3. 得票最多者出局
4. 重复以上步骤，直到满足胜负条件

### 胜负判定

- **平民获胜**：所有卧底出局
- **卧底获胜**：存活卧底人数 ≥ 存活平民人数
- **白板获胜**：白板猜出平民词和卧底词（可选规则）

## iPhone 添加到主屏幕

1. 用 Safari 打开游戏链接
2. 点击底部分享按钮
3. 选择「添加到主屏幕」
4. 即可像 App 一样全屏使用

## 部署

本项目部署在 [Render](https://render.com) 免费服务器上，访问链接即可使用。

如需自行部署：

```bash
git clone https://github.com/UTSJiyaoLi/who-is-spy.git
cd who-is-spy
npm install
cd server && npm install && cd ..
npm run build
npm run server
```
